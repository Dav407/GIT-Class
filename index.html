basic git commands

  1. git init: This is use to initialize a git repository
  2. git status: this is use to check the status of our file
  3. git add . : this is use to track the changes made to our file
  4. git commit -m "the actual commit message ": this is to save a snapshot of our file, using a descriptive caption
  5. git remote add origin: this is use to add a local repository to a remote repository on github
  6. git push origin main: this is use to push code from local to remote repository
